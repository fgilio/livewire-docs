# livewire-docs - Setup

## Install Globally

```bash
./install
```

Or manually:
```bash
ln -sf ~/.claude/skills/livewire-docs/livewire-docs ~/.local/bin/livewire-docs
```

## Verify

```bash
which livewire-docs && livewire-docs --version
```

## Uninstall

```bash
rm ~/.local/bin/livewire-docs
```
